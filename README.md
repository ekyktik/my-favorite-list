# RentHome - Вебдодаток для оренди житла

## Про проєкт
RentHome — це сучасна платформа для пошуку та оренди нерухомості. Застосунок дозволяє користувачам переглядати доступні варіанти житла, фільтрувати їх за ціною та локацією, а також зв'язуватися з орендодавцями.

**Технологічний стек:**
- **Backend:** Java (Spring Boot)
- **Frontend:** Angular
- **Database:** PostgreSQL
